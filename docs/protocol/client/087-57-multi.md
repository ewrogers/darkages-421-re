# `CMulti` (`0x57`)

[Previous: CSendPortrait](079-4f-send-portrait.md) | [Client action index](../client-actions.md) | [Next: baram](098-62-baram.md)

`CMulti` is the supplied later-client message name for client-direction action `0x57`. It is not RTTI or a symbol recovered from the 4.21 executable.

**Direction:** client to server

**Encrypted:** Yes. See [Sequence and XOR Transformation](../xor-transformation.md).

Payload offsets begin with the first byte after the action. The frame marker, frame length, action, sequence, and trailing zero are excluded.
