# Protocol Catalog

This catalog will describe the transport and link to one page per understood client or server message.

Each message page must explain what the message does, then record direction, opcode or discriminator, framing, field layout, byte order, string encoding, optional fields, state prerequisites, side effects, validation, and relevant function addresses.

## Transport status

Transport, framing, serialization, and XOR behavior have not been mapped yet.
